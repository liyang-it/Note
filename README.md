# 笔记日志

使用 [vitePress](https://vitepress.dev/zh)搭建

UI组件库使用[NaiveUI](https://www.naiveui.com/zh-CN/os-theme/docs/vitepress)

# 目录说明
```markdown
├── docs                    # 根目录
│ ├── .vitepress            # 配置文件夹
│ ├── public                # 资源文件夹
│ ├── *.md                  # Markdown源文件
├── node_modules            # NPM依赖
```

# 开始

## 安装依赖
```bash
npm i
```

## 启动
```bash
npm run dev
```
## 打包
```bash
npm build
```
